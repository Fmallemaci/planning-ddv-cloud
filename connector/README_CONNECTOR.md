# Planning DDV Outlook Connector

Instalacion en cada PC Windows:

1. Descargar y descomprimir `planning_ddv_outlook_connector.zip`.
2. Ejecutar `Instalar Planning DDV Outlook Connector.cmd`.
3. Volver a Planning DDV y presionar `Enviar por Outlook`.

El conector registra el protocolo `planningddv://` para abrir Outlook de escritorio con el perfil local del usuario.
